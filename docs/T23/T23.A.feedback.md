【严重度】⚠️ 体验

【期望】T23.A 期望流程为账号创建后进入成员资料页；如果产品实际流程是“账号 -> 工作区 -> 成员资料”，T23.A 文档和页面引导需要保持一致，不能让测试路径进入缺失信息状态。

【自动化测试】【现象】在本机已有真实 LLM `.env` 配置时，直接运行后端全量测试出现 3 个失败：诊断接口测试期望 `mock/error`，实际读取到真实 provider 并返回 `ok`；Agent smoke 测试期望 fallback，实际真实 provider 成功导致 `used_fallback=False`。
【期望】测试应隔离本地真实 `.env`，单元/集成测试默认不受真实 provider 配置影响；需要真实 provider 的测试应单独标记或通过显式环境变量开启。
【严重度】⚠️ 体验
